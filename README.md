# 🎬 izleyebilirsin

Türkçe arayüzlü, ücretsiz film ve dizi streaming sitesi.

## Özellikler

- 🎬 Trending filmler ve diziler
- 🔥 Top 10 bu hafta
- 🎲 "Ne İzlesem?" rastgele seçici
- ❤️ Favoriler (localStorage)
- ⏯️ Devam et (izleme geçmişi)
- 📺 4 farklı video kaynağı
- ⌨️ Klavye kısayolları
- 📱 Mobil uyumlu

## Kurulum

```bash
npm install
npm start
```

Sunucu `http://localhost:3300` adresinde başlar.

## Deploy

Railway, Render veya herhangi bir Node.js hosting üzerinde çalışır.
`PORT` ortam değişkenini otomatik okur.
